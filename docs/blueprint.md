# **App Name**: Al Iman Digital

## Core Features:

- Dynamic Content Fetching: Integrated data connection to existing Google Apps Script endpoint to populate the carousel, about section, and contact details.
- Digital Registration Portal: Functional student enrollment form with validation for 10-digit NISN and 16-digit NIK.
- Intelligent File Handler: Converts photos (JPG/PNG) and documents (PDF) to Base64 strings for direct API transmission without server-side storage.
- Curriculum Recommendation AI Tool: An AI assistant that analyzes student data to recommend the most suitable 'Iqra' or 'Tahfidz' program level.
- Interactive Activity Gallery: A responsive masonry-style grid for displaying institutional milestones and student activities with high-quality modals.
- Dynamic Stat Cards: Real-time visualization of institutional data including total students, teachers, and specialty programs.
- Location Integration: Embeddable interactive mapping interface for navigating to the school campus.

## Style Guidelines:

- Primary color: Deep Emerald (#1B5E3C) to represent faith, nature, and growth.
- Background: Soft Mint White (#F2F7F4) for a clean, educational atmosphere that prevents eye fatigue.
- Accent: Sage Leaf (#5BA178) used for secondary UI elements to maintain a harmonic analogous feel.
- Headline font: 'Poppins' (sans-serif) for its contemporary, geometric, and welcoming personality.
- Body text: 'Inter' (sans-serif) for maximum legibility on technical data like NIK/NISN inputs.
- Card-based interface using clean gutters and 16px to 24px padding to separate sections like 'Tentang' and 'Gallery'.
- Smooth transitions for 'Back to Top' buttons and entrance animations for section titles when scrolled into view.
